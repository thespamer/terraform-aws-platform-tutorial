# Advanced Practical Examples

This folder turns the advanced roadmap from the main README into concrete examples.

Each subfolder is intentionally practical and can be copied into a real platform repository after review.

## Examples included

| Topic | Folder |
|---|---|
| Terragrunt for multi-environment orchestration | `terragrunt/` |
| HCP Terraform or Terraform Enterprise | `hcp-terraform/` |
| AWS Organizations and multi-account landing zone | `aws-organizations/` |
| SCPs and permission boundaries | `scp-permission-boundaries/` |
| Argo CD / GitOps for Kubernetes workloads | `argocd-gitops/` |
| OPA/Sentinel with corporate policies | `policy-as-code/` |
| Recurring drift detection | `drift-detection/` |
| Cost estimation with Infracost | `infracost/` |
| Pipeline with environment approvals | `environment-approvals/` |
| Modules published in a private registry | `private-registry/` |

> These are templates. Validate permissions, naming, account IDs, regions and cost impact before using in production.
